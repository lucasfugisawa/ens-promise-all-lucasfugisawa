Implementar o método Promise.all.

Sua função deve receber um array de promises e retornar uma única promisse que resolve quando todas as promisses do array resolverem. A promise retornada deve resolver com um array com o resultado de todas as promisses recebidas, na devida ordem.

Exemplo:

  promiseAll([promise1, promise2, promise3]).then(res => {
    console.log(res[0]);// resultado de promise1
    console.log(res[1]);// resultado de promise2
    console.log(res[2]);// resultado de promise3
  }));
Dicas:
  • criando uma nova Promise;
  • como funciona Promise.all;

OBS1: apresentar 2 soluções: com e sem async/await;

OBS2: não é preciso tratar casos de erro e catch;

OBS3: não utilizar Promise.all();
